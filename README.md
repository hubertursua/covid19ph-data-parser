# covid19ph-data-parser

Outputs the data from https://covid19ph.com/ to a JSON file.

## Data Source

Kindly note that covid19ph.com is a secondary source of information. All of their sources are listed on the website.

## Usage

Run:

```sh
npm i
npm start
```
